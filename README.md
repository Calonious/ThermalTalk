# ThermalTalk
Pyramid Technologis Thermal Printer API

PyramidNETRS232 is available on NuGet

    Install-Package ThermalTalk

## Supports
* Real Time Status Requests
* Reliance Imaging (Phoenix TBD)
* Basic Font Controls
